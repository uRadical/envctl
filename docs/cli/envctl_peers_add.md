## envctl peers add

Add a peer manually

### Synopsis

Add a peer by address.

Use this when mDNS discovery isn't available (e.g., across subnets
or when connecting via VPN/Tailscale).

Example:
  envctl peers add 192.168.1.50:7834
  envctl peers add alice-laptop.local:7834
